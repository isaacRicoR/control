TAILWIND RULES (LAYOUT ONLY)
========================
Tailwind is allowed ONLY for layout.

- gap

- overflow-*, truncate, whitespace-*, text-ellipsis

FORBIDDEN:
- bg-*
- text-*
- border-*
- p-*, m-*, space-*
- rounded-*
- shadow-*
- Any class that changes font, size, weight or color
